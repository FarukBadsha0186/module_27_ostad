# 📰 News Portal Project

## 📌 Project Description
This is a full-stack News Portal web application built using:

- React.js  
- Tailwind CSS  
- Node.js  
- Express.js  
- MongoDB  

The system allows users to:

✅ Register and Login  
✅ View Latest News  
✅ Create, Update, and Delete News  
✅ Profile Management  
✅ Top 6 Latest News Section  
✅ Protected Routes  

---

## 🚀 Features

- User Authentication using JWT  
- Profile Update System  
- News CRUD Operations  
- Top Latest News API  
- Responsive UI using Tailwind CSS  
- Protected Dashboard  

---

## 🛠 Technologies Used

### Frontend
- React.js  
- Tailwind CSS  
- Axios  
- React Router  

### Backend
- Node.js  
- Express.js  
- MongoDB  
- JWT Authentication  
- Bcryptjs
